# MediMind AI 💊

An AI-powered medication reminder and monitoring system.

## Features
- Medicine Reminder
- Email Alerts
- Browser Notifications
- AI Health Assistant
- Emergency SOS
- Medicine Tracking
- MongoDB Database
- Spring Boot Backend
- React Frontend

## Tech Stack
Frontend:
- React.js
- Tailwind CSS
- Axios

Backend:
- Spring Boot
- Java
- MongoDB

Tools:
- VS Code
- IntelliJ IDEA
- GitHub

## Run Frontend
cd frontend
npm install
npm run dev

## Run Backend
cd medimind-backend
./mvnw spring-boot:run
